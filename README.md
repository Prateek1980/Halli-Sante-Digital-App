Halli Santhe Digital App

Problem Statement:
Local artisans and small sellers often struggle to reach a wider customer base due to limited digital presence, lack of affordable marketplace access, and dependence on traditional selling methods. Customers also find it difficult to discover authentic local handmade products in one convenient platform. This creates a gap between local sellers and potential buyers, limiting business growth and accessibility.

Approach:
To solve this problem, we designed and developed a mobile-based marketplace application that connects local artisans directly with customers. The approach involved understanding user needs, designing an intuitive Android app interface, categorizing products for easy discovery, implementing secure user authentication, and creating product browsing and inquiry features to simplify the buying process.

Work Done:

Developed the Android application UI with a clean and user-friendly design.
Implemented user registration and login functionality using email authentication.
Created product category sections for organized browsing.
Added product search functionality for easy item discovery.
Built detailed product view pages with product information.
Integrated stock inquiry / SMS contact option for customer communication.
Worked with Android Studio and AI-assisted development tools during implementation.

Technologies Used:
Kotlin – Used for Android app development
XML – Used for designing the user interface layouts
Android Studio – Primary development environment for building and testing the application
Firebase Authentication – For user signup and login functionality using email/password authentication
Firebase Realtime Database / Firestore – For storing user and product-related data
Android SDK – For app functionalities, UI components, and device compatibility
GenAI / AI-assisted Development Tools – Used for development assistance, code suggestions, and faster implementation
Git / Version Control – For project tracking and code management

Final Conclusion:
The project successfully created a functional prototype of a hyper-local artisan marketplace that helps bridge the gap between local sellers and customers. It improved accessibility, product discovery, and digital reach for artisans while providing users with a simple and organized shopping experience. The project also provided valuable hands-on experience in Android app development, UI design, and real-world problem solving.
